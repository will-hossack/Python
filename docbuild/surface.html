Classes to represent optical surfaces

Surface Class¶

Abstract base class to give represent a general surface.

class optics.surface.Surface(pt=0.0, type=0, index=None)[source]¶

Base class for an arbitrary surface, need to be extended to be useful. This class defines.surface reference point, surface type refractive index of image size (may be None) and membership of optical group.

Parameters:
  • pt (vector.Vector3d or float) – the surface reference point, defaults to (0,0,0)
  • type (int) – the surface type (default to Clear 0)
  • index (optics.wavelength.RefractiveIndex) – the refractive index on the image side of the surface (default to None)

ImagePlane Class¶

Class to represent an image plane, this is the same as OpticalPlane but wuth additial size information. This affets the .draw() methodd.

class optics.surface.ImagePlane(pt=0.0, xsize=36.0, ysize=24.0)[source]¶

Class to form an image plane (either input or output) of specific location and size.

Parameters:
  • pt (Vector3s or flrat) – the surface point,(Default to (0,0,0))
  • xsize (float) – the horizontal size, Defaults to 36.00 (horizontal size of 35 mm film)
  • ysize (float) – the vertical size, defaults to 24.00 (horizontal size of 35 mm film)

Note the size does NOT affect the surface interaction, it only alters the .draw() method.

QuadricSurface Class¶

Class to represent a quadratic surface.

class optics.surface.QuadricSurface(pos, curve, epsilon, radius, index=None)[source]¶

Method to implement a quadric surface determined by curvature and epsilon, the qudric factor

Parameters:
  • pos (Vector3d or float) – the plane reference point
  • curve (float) – the curcature.
  • epsilon (float) – the quadric parameter
  • radius (float) – the maxradius
  • index (RefractiveIndex or None) – the Refratcive index. If present, surface is refracting, if None it is reflecting.
