Material Classes¶

Classes to handle of materail database, mainly glasses, but also gases and liquids where are held in the asci databases using the same conventions as in RefractiveIndex.org. These class just handes the data bases of the materials, the refreatcive index is handeled by MaterialIndex() and InfoIndex() in the wavelangth module.

Material Class¶

class optics.material.Material(name, formula, wrange, coef)[source]¶

Class to hold a material type being name, formula and coefficents in the form held the same form as https://refractiveindex.info/

Parameters

  • name (str) – string name of material typically the key

  • formula (int) – formula type, note that formula 0 = invalid

  • wrange (list[low,high]) – range of validity of formula

  • coef (list[float]) – list of float coefficeint is same syntax as RefrativeIndex.info
