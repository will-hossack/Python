Ray Class¶

The base class in the Ray, which is abstract with the useful extensing classes being ParaxialRay and Intensity ray.

class optics.ray.Ray(wavelength=0.55, intensity=1.0, index=None)[source]¶

Base Ray class which just hold wavelength, intensity and refractive index all other (useful) classes extend this Base class. This class has internal variables of

  • self.intensity the intensity as float
  • self.refractiveindex the current optics.wavelength.RefactiveIndex
  • self.monitor the RayMonitor to record ray progress.
Parameters:
  • wavelength (float) – wavelength in microns (defaults to package Default = 0.55um)
  • intensity (float or optics.wavelength.Spectrum) – the intensity (defaults to 1.0)
  • index (optics.wavelength.RefractiveIndex) – refrative index (Default = None)

propagateThrough(surface)[source]¶

Method to propagate a ray through surface, or list of ray surfaces. This is the main method that does most of the work, it first propagate the ray to the surface and then depending on the surface it will block, reflect or refract the ray depending on the type of surface. If the ray is refracted its refractive index is also updated.

Parameters:surface (optics.surface.Surface or list of Surfces.) – Surface or list of list(Surface), if list each one is dealt with in order.
Returns:bool true is passed through, false if blocked.

Normalled called via the “*=” operator.

RayPencil¶

The most useful and powerful class for handling rays is the RayPencil, beinng a list or rays that can be propagated through surfaces with the “*=” operator. There are also a set of powerful method that allow the creation of various types or RayPencils.

class optics.ray.RayPencil(*args)[source]¶

Parameters:*args – list of rays to be added to the Pancil (Default = None)
