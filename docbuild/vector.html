Vector¶

The vector module deals with two and three dimensional vectors.

class vector.Vector3d(x_or_v=0.0, y=0.0, z=0.0)[source]¶

Class to implement three dimensional vectors.

Parameters:
  • x_or_v (float, Vector3d, list) – x component, Vector3d r of a list of floats. (default = 0.0)
  • y (float) – y component (default = 0.0)
  • z (float) – z component (default = 0.0)

If x_or_v is a Vector3d or list then the valules of the y and z parameters will not be accessed.
