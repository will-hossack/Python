Vector Classes¶

These classes handle two and three dimensional vectors with many supporting methods to simplfy their use. These classes are used throughout the optics package but can also be used as stand alone classes for other applications.

Note these vector classes are all interally hand coded and do not use Numpy.

Unit3d Class¶

Extending class to handle unit three dimensional vectors, this class is typically used for ray directions.

class vector.Unit3d(x_or_v=0.0, y=0.0, z=0.0)[source]¶

Class to hold a unit Vector3d, it extends Vector3d with automormalsiation on creation and extra methods to support optical ray calcualtions.

Parameters

  • x_or_v – x componen, Angle, list[] (default = 0.0)

  • y – the y component (default = 0.0)

  • z – the z component (default = 0.0)

If parameter is NOT a Unit3d or Angle it is automatically normalsied to unit. Note if (0,0,0) or () suppled, the Unit3d will be set to inValid.
