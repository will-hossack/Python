Psf Class¶

Class to implment a sinmple geometric PSF being a being an elipse about a specified golobal coordinate position.

class optics.psf.Psf(pos=Vector3d (0.0000e+00, 0.0000e+00, 0.0000e+00), intensity=1.0, major=1.0, minor=None, alpha=0.0, wave=None)[source]¶

Class to represent a geometric Psf as the best fitting ellipse. There area also methods to allow this to be calcualted from a RayPencil in a specified OpticalPlane.

Parameters

  • pos (Vector3d or float) – centre of PSF (Default = (0,0,0))

  • intensity (float) – the intensity (Default = 1.0)

  • major (float) – major axis of ellipse (Default = 1.0)

  • minor – minor axis of ellipse (Default = None), major values used

  • alpha (float) – angle of ellipse (Default = 0.0)

  • wave – the wavelength, Default = None
