Wavelength Module¶

This module deal with the wavelnegth dependand features in particular refractive and intensity spectrums.

Wavelength Value¶

There are a set of globals defined covering the standard wavelength. All are in microns.

Set of classes to deal with optical wavelengths and functions associated with wavelengths including refractive index and spectra. It aslo handles the default wavelength of the package.

Define Abstract class to deal with functions of wavelength. There is a default constructor to set defaults, typically called by extending classes only there are no parameters, it just initialises variables.

CauchyIndex Class¶

Class to implement the simple Cauchy Index formula with either

  • the three Cauchy paramters, (a,b,c)
  • the Nd and Vd values
  • the 6 digit type integer XXXYYY where Nd = 1.XXX and Vd = Y.YY
class optics.wavelength.CauchyIndex(a_or_nd, b_or_vd=None, c=None)[source]¶

Class to implement the simple a + b/lambda^2 + c / lambda^4 Cauchy index, with either a,b,c or Nd,Vd or XXXYYY int whhere Nd = 1.XXX and Vd = Y.YY

Parameters:
  • a_or_nd (float OR int) – a vaule in the Cauchy formula if 3 parmeters, the Nd value if two, or type integer if one.
  • b_or_vd (float) – b value in Cauchy formula if 3 parameters, else Vd id two given
  • c (float) – c value in Caucby formula (may be None)

InfoIndex class¶

Class to implement the refractive index calculations in the format used by RefrativeIndex.info site, this is mainly interval class used by the MaterialIndex, AirIndex and CauchyIndex to do the actual calculations.

class optics.wavelength.InfoIndex(formula, wrange, coef, name='InfoIndex')[source]¶

Implment a RefractiveIndex in the format suppled by RefractiveIndex.info website, all have common calls.

Parameters:
  • formula (int) – formula type, (1,2,3,5,6 and 7 implementated)
  • wrange (list) – list of two float giving validity range
  • coef (list) – list of floats holding the coefficents.
  • name (str) – name or key (defaults to InfoIndex)

This class is not typically called by the used, the normal intreface is MaterialIndex
